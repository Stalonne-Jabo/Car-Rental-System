Car Rental System

A placeholder landing page for the Car Rental project.

About

This repository contains the early scaffolding for a car rental application with a Node/Express backend and a React frontend. The client application is still under active development.

Planned features

  • Browse and search available rental cars
  • Book a car for a chosen date range
  • Manage bookings as a customer
  • Owner dashboard to list and manage vehicles
  • User registration and authentication

Deployment notes

Netlify is a static hosting and serverless platform, so the long-running Express server in server.js is not deployed here. Once the React client is complete, build output will replace this placeholder as the published site, and backend routes can be migrated to Netlify Functions.
